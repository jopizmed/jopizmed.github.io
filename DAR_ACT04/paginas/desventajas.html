DESVENTAJAS

Para detallar las desventajas usaremos las clases previstas en Bootstrap para configurar los diferentes botones partidos con "dropdown".


Todos los diseños hechos con Bootstrap son similares Aunque se pueden modificar los css de Bootstrap Pero el trabajo sería demasiado costoso
Aunque inicialmente tiene una buena curva de aprendizaje Pero llegar a dominarlo puede ser bastante costoso
Para determinados usos puede ser demasiado pesado Esto puede hacer que se ralenticen los tiempos de carga
Si necesitamos algún componente nuevo hay que hacerlo de cero Puede ser dificil adaptarlo al "look" predefinido de Bootstrap
