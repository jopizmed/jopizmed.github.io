VENTAJAS

Para detallar las ventajas usaremos las clases previstas en Bootstrap para configurar los diferentes tipos de alerta.


1. Fácil de usar Comenzar con Bootstrap es un procedimiento extremadamente fácil y rápido. Es muy adaptable, y su curva de aprendizaje es bastante suave
2. Capacidad de respuesta Cada año, los dispositivos móviles se hacen vez más populares, y el requisito de tener un sitio web responsivo se ha vuelto obligatorio. A medida que el tamaño de la pantalla varia, el diseño de la web se adapta de forma automática a los distintos tamaños. Crear un sitio listo para dispositivos móviles es una tarea fácil y sencilla con Bootstrap. Con el uso de clases ya preparadas de Bootstrap, se puede reconocer el número de puntos en el sistema de cuadrícula que queremos que ocupe cada columna.
3. Velocidad de desarrollo Uno de los principales beneficios de utilizar Bootstrap es la velocidad del desarrollo. En lugar de codificar desde cero, con Bootstrap se pueden usar bloques de código ya listos para programra un sitio web. Además, con la funcionalidad CSS-Less y la compatibilidad entre navegadores hacen que sea una estratégia adecuada para ahorrar muchas horas de entrada de código. Incluso se pueden comprar temas Bootstrap ya preparados y modificarlos para que se ajusten a nuestro diseño.
4. Personalizable Bootstrap se puede personalizar según las necesidades de nuestro diseño. Al desarrollar una web se puede partir de la version original de los diseños de componentes de Bootstrap e ir modificando y adaptando a las necesidades de nuestro diseño, css, tipografías comportamientos de JavaScript, menus...etc.
5. Apoyo Bootstrap posee una gran comunidad de usuarios en internet. En esta comunidad se resuelven la mayoría de problemas de diseño y funcionalidad a la hora de diseñar con Bootstrap. Actualmente, el código de Bootstrap está alojado en GitHub y tiene más de 500 desarrolladores directos.
6. Integracion Simple Puede integrarse de forma muy sencilla con otras plataformas, en sitios web que ya existen o en desarrollos nuevos. Además puede utilizarse parcialmente, son solo aquellos componentes de diseño que nos interesan.
7. Componentes prediseñados Viene ya con numerosos componentes prediseñados para mostrar alertas, menus desplegables, barras de navegación, que son facilmente integrables y adaptables a nuestro diseño.
